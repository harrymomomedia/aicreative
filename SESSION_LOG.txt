PHASE 13 — CAPTIONS
-------------------------------------------------------------------------------

USER: "how about adding subtitle, how can we add subtitle? that is
beautiful?"

ASSISTANT presented options (Whisper+ffmpeg DIY / CapCut / Submagic /
Captions.ai / Descript). USER picked DIY in-pipeline first.

Built caption.py:
  - Extract audio
  - Whisper word-level transcribe
  - Chunk words into phrases (max-words + natural pauses >0.35s)
  - Render PNG per chunk via PIL
  - Overlay via ffmpeg

DEAD END: ffmpeg subtitles= filter
  - Homebrew ffmpeg 8.1 was built without --enable-libass
  - "No such filter: 'subtitles'" error
  - Pivoted to PIL renderer + overlay filter

DEAD END: HelveticaNeue.ttc index=8
  - Loaded an italic variant, rendered with weird zebra-stripe outlines
  - Switched to Arial Black .ttf (single-face TTF, no index ambiguity)
  - Used PIL's native stroke_width= for clean outlines

USER: "it's chowchilla, chino and folsum, and the subtitle is too big and
ugly we need it to look more like submagic one, smaller font and also
max 2 rows, and spacing between rows also need to be reasonable."

Found Whisper mistranscriptions:
  Chowchilly → CHOWCHILLA
  Falsum → FOLSOM
  Chino → already correct

Added SUBSTITUTIONS dict to caption.py for proper-noun fixes.

Submagic-style tuning:
  fontsize_ratio:  0.062 → 0.045 → 0.035 (3.5% of frame height)
  outline_ratio:   0.005 → 0.003 (and now scales with fontsize)
  marginv_ratio:   0.20 → 0.16
  line_gap:        15% of fontsize → 8%

USER: "and significant potential shows 4 rows still."

Issue: my word-wrap had no max-line cap. Long-word chunks like
"SIGNIFICANT POTENTIAL FINANCIAL COMPENSATION" would overflow to 4 lines.

Added adaptive font shrink — if wrap exceeds max_lines (default 2),
shrink fontsize ~8% and retry up to 10 iterations or fontsize 18px floor.

  final_v7_captioned.mp4 — final deliverable

KEY DECISIONS LOG
-------------------------------------------------------------------------------

1. Free local Whisper > OpenAI API for dissect transcription
2. Same character anchor across 3 clips (character consistency over
   frame-to-frame continuity)
3. Survivor voice + kitchen character (matched visual to script tone)
4. "Chow-chilluh" phonetic respell (vs space-separated "Chow chilla")
5. Lawyer firm voice script with "potential" hedge for legal compliance

9. Show character first (no b-roll at 0-4s) for identity establishment
10. PIL+overlay captioning (libass not available on Homebrew ffmpeg)
11. TikTok/Submagic font sizing: 3.5% of frame height with adaptive shrink
12. Substitution dict for Whisper proper-noun mistranscriptions
